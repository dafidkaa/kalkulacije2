# OG Image Generation Guide

## Template Created
A base OG image template has been generated at: `og_image_template.png`

## How to Create Calculator-Specific Images

### Option 1: Use Canva (Recommended)
1. Upload the template to Canva
2. Duplicate for each calculator
3. Add calculator-specific text overlay at the top:
   - "Kalkulator Plaće" 
   - "BMI Kalkulator"
   - "Kalkulator Postotaka"
   - etc.
4. Export as PNG (1200x630px)
5. Save to `/public/og-images/[calculator-name].png`

### Option 2: Use Figma
1. Import template
2. Create variants for each calculator
3. Export all as PNG

### Option 3: Automated (Future)
Consider using a service like:
- Cloudinary Dynamic Images
- imgix
- Vercel OG Image Generation
